# base-batis
Based on myBatis，单表操作更便捷。

#### dependency
加入maven dependency即可，无需其他配置
```xml
<dependency>
    <groupId>dominic.base</groupId>
    <artifactId>base-batis</artifactId>
    <version>0.3.0</version>
</dependency>
```
